Restriction of a substitution to a term.

\begin{code}
R = Σ × Λ
\end{code}

Freshness in a restriction is defined as follows:

\begin{code}
_#⇂_ : V → R → Set
x #⇂ (σ , M) = (y : V) → y * M → x # (σ y)
\end{code}


Free

\begin{code}
_*⇂_ : V → R → Set
x *⇂ (σ , M) = ∃ (λ y  → (y * M) × (x * σ y))
\end{code}
