    • Formal Metatheory of λ-calculus

    We develop metatheory of the λ-calculus in Constructive Type Theory, which is made possible by the use of an appropriate notion of substitution, due to A. Stoughton.

    Project maintained by ernius Hosted on GitHub Pages — Theme by mattgraham

    We develop metatheory of the Lambda calculus in Constructive Type Theory, using the presentation of the former with one sort of names for both free and bound variables and without identifying terms up to α-conversion. We prove four substitution lemmas: for α-conversion, β-reduction, parallel β and for the system of simple type assignment. This work prove the main results of the metatheory; subject reduction of the Type System, and Church-Rosser confluence theorem. This results are possible by the use of an appropriate notion of substitution, due to A. Stoughton. The whole development has been machine-checked using the system Agda. The whole source code is web browsable [here](http://ernius.github.io/formalmetatheory-stoughton/html/index.html).
